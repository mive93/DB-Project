HOW TO USE OUR SOFTWARE

Estrarre l'archivio.
Aprire PgAdmin III e creare un nuovo datatbase chiamato "ProgettoMartoglia".
Aprire l'SQL editor poi aprire il file CreazioneDatabase.sql e lanciarlo.
Aprire la command line e spostarsi nella cartella src.

Eseguire il comando di Windows : java -classpath .;postgresql.jar ProgettoMartoglia
Eseguire il comando su Linux : java -classpath .:postgresql.jar ProgettoMartoglia

Inserire usr e psw di postgres per utilizzare il software.
Enjoy it!
